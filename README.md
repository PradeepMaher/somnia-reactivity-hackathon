# somnia-reactivity-hackatho

## 📌 Project Title
Reactive AI Wallet Monitor

## 🧠 Problem Statement
Blockchain users lack real-time intelligent monitoring of suspicious or unusual wallet activities.

## 💡 Solution
A reactive Web3 dashboard that:
- Monitors wallet transactions in real time
- Detects unusual patterns
- Sends alerts instantly

## 🛠 Tech Stack
- React.js
- Node.js
- Solidity
- Web3.js
- AI-based anomaly detection

## 🎯 Features
- Connect wallet
- Live transaction updates
- Smart alerts
- Clean dashboard UI
